# World Instrument

## Simulate on Unity Editor

- Change the platform to iOS and Play `Scenes/Onboarding` or `Scenes/Performance`.

## How to build

- Enter your Google Geospatial API key to `ProjectSettings/ARCoreExtensionsProjectSettings.json`
